## What is Peek?

Peek is a browser extension that brings information to you — the moment you need it. Select any word, phrase, or image on a webpage and a smart popup appears instantly, surfacing relevant definitions, summaries, and answers without ever leaving the page.

No new tabs. No copy-pasting into search bars. Just select and know.

## What can you select?

- **Text** — words, phrases, technical terms, anything written
- **Images** — get context around what you're looking at
- **Google Forms** — understand form fields and questions at a glance

## What Peek shows you

When you select something, Peek pulls from multiple trusted sources and presents them together in one clean popup:

- **Wikipedia** — quick summaries for people, places, concepts, and more
- **Wiktionary** — clear definitions and word meanings
- **DuckDuckGo Instant Answers** — fast facts and direct answers
- **Stack Overflow** — explanations for technical and programming terms

Every result comes with a source link so you can dive deeper, and a one-click **Copy** button to grab what you need.


## Install

1. Open **Chrome** or **Edge** and go to the Extensions page
2. Turn on **Developer mode**
3. Click **Load unpacked**
4. Select the Peek folder

That's it — Peek is ready to use on any webpage.


## How to use

1. **Select** any word, phrase, or image on a page
2. **Read** the instant popup with summaries and definitions
3. **Click** a source link to learn more, or hit **Copy** to grab the text
